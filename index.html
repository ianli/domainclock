About

Domain Clock generates domain names of the current time. I created Domain Clock after browsing a web site made of four numbers (I don't remember the URL), and realizing that the numbers represent time with digits for hours and minutes.

Implemented in PHP, HTML, CSS, and Javascript. Used the jQuery Javascript library. The font for the body is Helvetica. The layout is built on top of Blueprint CSS.
